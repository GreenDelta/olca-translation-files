Novedades y mejoras

Para la versión 1.4 hemos rediseñado y "renovado" completamente openLCA. Los resultados son:

  • drástico incremento del rendimiento de alrededor del 400%
  • numerosas mejoras de uso
  • muchas nuevas funcionalidades

¿Qué novedades hay en openLCA 1.4.1? Averígualo aquí.

Cómo empezar

Es realmente fácil empezar a usar openLCA en la nueva versión. Sólo hay que importar una base de datos desde un archivo zolca existente, y después abrirla haciendo doble click sobre ella en el menú de navegación de la izquierda.

Algunas funcionalidades básicas de openLCA se explican también aquí.

Buscar y descargar datos

Un gran número de bases de datos de ACV están disponibles en openLCA Nexus, donde se pueden buscar tanto series de datos individuales como bases de datos completas. Todas estas bases de datos pueden utilizarse directamente en openLCA, incluso combinándolas entre ellas.

Paquetes de métodos para la evaluación del impacto del ciclo de vida (EICV) están disponibles en la página de descargas de openLCA.

Hay una explicación más detallada de cómo importar bases de datos y métodos para EICV en openLCA aquí.
