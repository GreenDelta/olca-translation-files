Novedades y mejoras

Para la versión 1.4 hemos rediseñado y "renovado" completamente openLCA. Los resultados son:

  • drástico incremento del rendimiento de alrededor del 400%
  • numerosas mejoras de uso
  • muchas nuevas funcionalidades

Cómo empezar

Cuando se inicia openLCA por primera vez, no contiene ningún dato. Pero es realmente fácil empezar a usar openLCA. Sólo hay que importar una base de datos desde un archivo zolca existente. Después, abra la base de datos haciendo doble click sobre ella en el menú de navegación de la izquierda.

Ésto se explica más detalladamente en un vídeo disponible aquí

Buscar y descargar datos

Un gran número de bases de datos de ACV están disponibles en openLCA Nexus, donde se pueden buscar tanto series de datos individuales como bases de datos completas. Todas estas bases de datos pueden utilizarse en openLCA, incluso combinándolas entre ellas.
