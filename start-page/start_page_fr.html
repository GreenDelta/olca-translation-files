Quoi de neuf ?

Pour la version 1.4, nous avons complètement redessiné et "rénové" openLCA. Résultat :

  • des performances drastiquement améliorées, jusqu'à 400%
  • de nombreuses améliorations pour vous faciliter la vie
  • beaucoup de nouvelles fonctionnalités

Mise en route

Quand vous lancez openLCA pour la première fois, il ne contient aucune donnée. Avec la nouvelle version d'openLCA, il est très facile de se lancer. Vous pouvez simplement importer une base de données à partir d'un fichier zolca existant par un simple clic droit dans la Navigation, à gauche de la fenêtre. Ensuite, double-cliquez sur la base de données pour l'ouvrir.

Vous pouvez vous aider de ce tutoriel vidéo

Trouver et télécharger des données

Un large éventail de bases de données d'ACV sont disponibles sur openLCA Nexus, où vous pouvez trouver des données uniques ou des bases de données complètes, et les charger dans openLCA. Ces bases de données sont toutes adaptées pour openLCA, et vous pouvez en combiner plusieurs.
