入门指南

当第一次启动 openLCA, 没有数据. 在新版本, 是真的很容易就可以开始. 你可以简单地从现存的数据库输入 zolca 文件, 然后在导航中双击数据库名称以开启

搜索和下载数据

许多不同的数据库是可在 openLCA Nexus取得, 那里你可以搜索的完整数据库，单个数据集或下载至 openLCA使用. 各数据库均适合 openLCA, 而且可以结合多个数据库.
